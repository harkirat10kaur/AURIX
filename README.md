# AURIX – College Event Management & Collaboration Platform

AURIX is a full-stack web application developed to streamline college event management and student collaboration through a centralized platform.

The platform enables students, club admins, and system administrators to manage events, announcements, registrations, and club activities using role-based access.

---

## Features

* Student registration and login
* Event creation and management
* Club announcements
* Event registrations
* Role-based dashboards
* Admin management system

---

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MySQL

---

## Installation & Setup

### Clone the Repository

```
git clone https://github.com/harkirat10kaur/AURIX.git
```

### Navigate to Project Folder

```
cd AURIX
```

### Install Dependencies

```
npm install
```

### Setup Database

```
node setup_db.js
```

### Run the Server

```
node server.js
```
